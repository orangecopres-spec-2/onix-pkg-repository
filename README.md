# onix-pkg-repository
The package repository for onix the orangeco-os
the onix package format will be .o.pkg and i will provide the script to decode them on this repo when i finish
so to use the scriplt to install you do ./opkg install example.o.pkg and to create a demo pkg ./opkg create example 1.0 ./opkg and if you want to use in in an os check out onix-kernel the onix orangeco operating system
